About

CinemaBook is a modern movie ticket booking web application designed to provide a smooth and engaging user experience. Users can explore movies, choose their preferred language and show timings, select seats visually, and generate tickets instantly.

This project demonstrates key web development concepts such as dynamic page navigation, responsive UI design, and client-side data storage using LocalStorage.

Our goal is to make movie booking simple, fast, and enjoyable for everyone 🎬
